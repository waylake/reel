## 제품 설명 (Gumroad 페이지용)

### 요약 (상단)

Reel은 메뉴바에 상주하는 macOS 미디어 다운로더다. 브라우저에서 링크를 복사하면 자동으로 감지하고, yt-dlp 엔진으로 다운로드한다. SponsorBlock, 자막 임베드, 병렬 다운로드를 별도 설정 없이 사용할 수 있다.

### 기능

**클립보드 자동 감지**
링크를 복사하면 메뉴바에서 즉시 인식한다.

**병렬 다운로드**
최대 6개 동시 다운로드. 저전력 프로필로 배터리 사용량을 줄일 수 있다.

**오디오 추출**
MP3, M4A 프리셋으로 오디오만 뽑아낸다. 팟캐스트, 강의, 음악에 적합.

**광범위한 사이트 지원**
YouTube, Vimeo, TikTok, X, Instagram, Reddit, Twitch, SoundCloud, Bilibili 등 yt-dlp가 지원하는 1,800개 이상의 사이트에서 동작한다.

**SponsorBlock**
스폰서, 자기홍보, 상호작용 구간을 자동으로 건너뛴다.

**자막과 메타데이터**
다국어 자막 임베드, 챕터 마크, 썸네일, 메타데이터를 영상에 포함한다.

**큐 영속화**
앱을 종료해도 대기 중인 다운로드가 유지된다. 재시작 시 자동 복원.

## 구매자 안내

### 설치 방법

1. `Reel.app.zip` 압축 해제
2. `Reel.app`을 Applications 폴더로 이동
3. 실행 — 메뉴바에 ↓ 아이콘이 나타난다

macOS 보안 경고가 뜨는 경우:
- 시스템 설정 → 개인정보 보호 및 보안 → '그대로 사용' 클릭
- 또는 Reel.app 우클릭 → 열기 → 확인

### 처음 사용할 때

- 브라우저에서 링크를 복사하면 자동 감지된다
- ⌘V로 붙여넣어 바로 다운로드할 수 있다
- ⌘, (설정)에서 저장 위치와 기본 프리셋을 변경할 수 있다
